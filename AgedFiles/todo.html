Pick the shared folder

Pick a folder on a shared drive (OneDrive, Dropbox, Google Drive Desktop, mapped network drive, etc). The app will create data.json, README.txt, and a backups/ subfolder there. Anyone who opens this same HTML file and points to the same folder will see your edits within ~3 seconds.

⚠ Your browser doesn't support the File System Access API (needs Chrome / Edge / Opera). You can still use the app locally — use Backup / Restore JSON to share manually.
